# 🚀 快速开始指南

## 5分钟部署 AI 拍照识物小程序

### 第一步：下载并导入项目

1. 下载微信开发者工具：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 使用微信开发者工具打开本项目目录
3. 扫码登录微信开发者账号

### 第二步：配置 AppID

1. 在微信开发者工具中，点击右上角"详情"
2. 在"基本信息"中修改 AppID 为你的小程序的 AppID
3. （测试阶段可使用测试号）

### 第三步：开通云开发

1. 在微信开发者工具中，点击右上角"云开发"
2. 开通云开发环境
3. 选择按量付费或包年包月（推荐按量付费）

### 第四步：上传云函数

1. 右键点击 `cloudfunctions/classify` 目录
2. 选择"上传并部署：云端安装依赖"
3. 等待上传完成

### 第五步：运行测试

1. 点击微信开发者工具中的"编译"按钮
2. 在模拟器中测试功能：
   - 点击"启动摄像头"
   - 拍照识别
   - 查看识别结果

## 🎯 集成真实 AI 识别（可选）

### 使用腾讯云 API（推荐）

1. 注册腾讯云账号：https://console.cloud.tencent.com/
2. 开通"图像识别"服务
3. 获取 SecretId 和 SecretKey
4. 修改 `cloudfunctions/classify/index.js`，替换为真实 API 调用代码
5. 重新上传云函数

详细步骤请参考：`AI_API_GUIDE.md`

## 🐛 常见问题

### Q: 摄像头无法启动？
A: 检查小程序权限配置，确保已授权摄像头权限。在真机上测试效果更好。

### Q: 识别结果为模拟数据？
A: 这是预期行为。需要集成真实的 AI API 才能识别真实图片。

### Q: 云函数部署失败？
A: 检查网络连接，确保云开发环境已开通。
